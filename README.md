# nicegui_keyboard
Smart keyboard for NiceGUI

<img width="1898" height="387" alt="Screenshot 2025-11-28 144759" src="https://github.com/user-attachments/assets/55fd1434-3da8-4ed8-9a4f-6cb1cbfebfb4" />

## Use:
```python
from keyboard import VirtualKeyboard

keyboard = VirtualKeyboard()

keyboard.attach(ui.input("Type text"))        

ui.run()       


```
